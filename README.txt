Count any post views per user per date.

== Description ==

The plugin helps you to track the post view count. It simply shows the post view count per user per date. You can also export it in CSV format. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/user-login-history` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. To see the report, just visit `Admin Dashboard >> Users >> Count Post Views`.

== Frequently Asked Questions ==
= Can this plugin track the post view count of guest users? =
No. The plugin does not track the post view count of guest users.

= Is it compatible with WordPress multisite network? =
Not yet. We are working on this feature.

= Is there any shortcode to show the report on the frontend? =
No. The plugin does not have any shortcode.
